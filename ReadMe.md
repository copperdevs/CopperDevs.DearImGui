# Coppers DearImGui

> Custom layering around [ImGui.NET](https://github.com/ImGuiNET/ImGui.NET)/[DearImGui](https://github.com/ocornut/imgui)

## Info
- This is currently just a wrapper for existing DearImGui bindings, so ImGui.NET is required
